# Floating Text

A Unity package for creating floating text popups — damage numbers, score popups, item pickups, and any other temporary text that floats upward and disappears.

## Installation

Add to your `Packages/manifest.json`:

```json
"com.alexandaar.floatingtext": "https://github.com/Hunter13ua/FloatingText.git"
```

Or for local development:

```json
"com.alexandaar.floatingtext": "file:Packages/com.alexandaar.floatingtext"
```
